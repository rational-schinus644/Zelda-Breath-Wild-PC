## 📑 Troubleshooting Common Issues

Read these solutions if you run into trouble during the setup or play process.

**Game does not start**
Verify that your PC meets the listed requirements. Check that your Windows installation has all current updates.

**Visuals appear blurry**
Open the Options menu and select Graphics. Check that the resolution matches your monitor settings. Make sure High Resolution Textures are set to Enabled.

**Input lag occurs**
Connect your controller with a USB cable if you notice a delay. Wireless connections sometimes impact input speed.

**The game crashes during startup**
Close background programs like browsers or video capture tools. These programs compete for memory and graphics power. Restart your PC to clear your system memory before you try again.
